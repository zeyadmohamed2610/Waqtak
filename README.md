# ⏱️ وقتك | Waqtak — Bybit Account Tracker

**وقتك (Waqtak)** هو تطبيق ويب متطور، ذكي وآمن بالكامل، مصمم خصيصاً لمتابعة حسابات وتتبع فترات المراجعة والجاهزية لإيداعات منصة Bybit خطوة بخطوة. يتميز التطبيق بواجهة مستخدم زجاجية عصرية (Glassmorphic UI) تفاعلية بالكامل، تدعم اللغتين العربية والإنجليزية ديناميكياً وتعمل بالكامل على جانب العميل (Client-Side) لضمان خصوصية وأمان البيانات بنسبة 100%.

---

## ✨ الميزات الرئيسية | Key Features

### 1. 🔔 نظام الإشعارات الذكي (Browser Notifications)
* يرسل تنبيهات فورية لسطح المكتب أو الهاتف عند انتهاء المرحلة الأولى (جاهز للمهمات) وعند اكتمال المرحلة الثانية (جاهز بالكامل).
* يقوم بجدولة الإشعارات المستقبلية تلقائياً لكل حساب.
* يتحقق عند فتح التطبيق من أي إشعارات فائتة (انتهت مؤقتاتها أثناء إغلاق الصفحة) ويرسلها فوراً للمستخدم.

### 2. 🛡️ التخزين الثنائي المحمي (Dual Storage Backup)
* لحماية البيانات من الحذف المفاجئ، يقوم التطبيق بحفظ الحسابات في مسارين منفصلين تزامناً:
  * **`localStorage`**: للتحميل الفوري والسريع للبيانات.
  * **`IndexedDB`**: قاعدة بيانات متصفح محلية عميقة ومنفصلة تماماً.
* **Data Integrity Panel**: أداة فحص بصرية تقارن بين الـ localStorage والـ IndexedDB وتعرض الفروقات وتسمح بالمزامنة والاستعادة بنقرة واحدة.

### 3. 📊 لوحة الإحصائيات (Stats Dashboard)
* ملخص لحظي لكافة الحسابات يوضح:
  * إجمالي عدد الحسابات المضافة.
  * الحسابات النشطة في المرحلة 1 والمرحلة 2.
  * الحسابات المكتملة كلياً.
  * **إجمالي مبالغ الإيداع الكلي** منسقاً حسب لغة الواجهة.
  * عدد الحسابات التي ستنتهي **اليوم** (المتبقي لها أقل من 24 ساعة).

### 4. 📈 الجدول الزمني المرئي (Timeline Visualization)
* عرض بصري (Gantt-like) لكل الحسابات على خط زمني واحد ملون.
* شريط ذهبي للمرحلة 1 (4 أيام) + شريط برتقالي للمرحلة 2 (4 أيام).
* خط "اليوم" الأحمر يوضح أين أنت الآن بالنسبة لكل حساب.
* النقر على أي حساب يفتح البطاقة المقابلة.

### 5. ☑️ التحديد المتعدد والعمليات الجماعية (Bulk Actions)
* وضع التحديد المتعدد — ظلل عدة حسابات دفعة واحدة.
* **حذف جماعي** — احذف حسابات متعددة بنقرة واحدة.
* **تصدير جماعي JSON** — صدّر الحسابات المحددة كملف احتياطي.
* **تصدير CSV** — صدّر كجدول Excel للقراءة والتقارير.

### 6. 📋 أزرار النسخ السريع والعمليات (Quick Action Controls)
* زر نسخ سريع بجانب كل من الـ UID، البريد الإلكتروني، وعنوان الـ IP.
* سجل تغييرات تاريخي (Audit Log) مدمج في كل بطاقة.

### 7. 💾 تصدير واستيراد البيانات (JSON Backup & Restore)
* **التصدير**: بضغطة زر، تحميل ملف `.json` احتياطي بالاسم والتوقيت.
* **الاستيراد**: إمكانية رفع ملف النسخة الاحتياطية لاستعادة الحسابات.

### 8. 🏷️ التصفية والفرز المتقدم (Filtering & Sorting)
* فلاتر مخصصة: الكل / المرحلة 1 / المرحلة 2 / مكتمل.
* فرز: الأحدث إيداعاً / الأقدم إيداعاً / الأقرب للانتهاء.

### 9. ⌨️ اختصارات لوحة المفاتيح (Keyboard Shortcuts)

| الاختصار | الوظيفة |
|---------|---------|
| `Ctrl + Enter` | احسب / احفظ |
| `Ctrl + N` | حساب جديد |
| `Ctrl + E` | تصدير البيانات |
| `Ctrl + B` | تبديل الوضع (فاتح/داكن) |
| `Ctrl + K` | فتح قائمة الاختصارات |
| `Ctrl + Shift + F` | بحث سريع |
| `Esc` | إغلاق النموذج |
| `?` | فتح دليل الاختصارات |

### 10. 🌙 الوضع الفاتح / الداكن (Light/Dark Mode)
* تبديل فوري بين الوضع الداكن (افتراضي) والوضع الفاتح.
* يُحفظ التفضيل تلقائياً.

### 11. 📱 تطبيق PWA — يعمل أوفلاين
* قابل للتثبيت كتطبيق على الجوال والكمبيوتر.
* يعمل بدون إنترنت عبر Service Worker.
* رسالة "ثبّت التطبيق" تظهر تلقائياً عند دعم المتصفح.

## 💻 أنماط التشغيل | Operational Modes

### 1. الحاسبة السريعة (Quick Calculator Mode)
* مخصصة لحساب فترات الـ 4 أيام والـ 8 أيام بشكل فوري لحساب واحد دون حفظه.
* تدعم 3 طرق لإدخال التاريخ:
  * **تنسيق UTC**: مثل `2026-05-26 11:36:42 (UTC+0)`.
  * **تنسيق بريد Gmail**: مثل `May 26, 2026, 2:36 PM` ويتم معالجته وتوطينه تلقائياً.
  * **تنسيق يدوي**: اختيار التاريخ والوقت بالتوقيت المحلي.

### 2. مدير الحسابات الذكي (Account Manager)
* مفكرة ذكية لإدارة حسابات متعددة.
* زر **"الآن ⏱️"** لتعبئة الوقت الحالي تلقائياً.
* مؤقتات عد تنازلي حية تعمل ثانية بثانية لكل بطاقة.

## 🛠️ التكنولوجيا المستخدمة | Tech Stack

* **Structure**: HTML5 (Semantic Elements)
* **Styling**: Vanilla CSS3 (Custom Variables, Flexbox, CSS Grid, Glassmorphism, Backdrop Filters, Light/Dark Themes)
* **Logic**: Vanilla JavaScript (ES6+, Promises, Async/Await)
* **APIs**:
  * **IndexedDB API**: للتخزين الاحتياطي المحلي طويل المدى.
  * **Service Worker API**: للعمل أوفلاين وتثبيت PWA.
  * **Web Notifications API**: لإرسال الإشعارات.
  * **Clipboard API**: للنسخ السريع بنقرة زر.

## 📦 التثبيت والتشغيل | Installation

1. Clone المستودع:
   ```bash
   git clone https://github.com/zeyadmohamed2610/Waqtak.git
   ```
2. افتح مجلد المشروع:
   ```bash
   cd Waqtak
   ```
3. افتح `index.html` مباشرة في أي متصفح، أو ارفعه على **Vercel** أو **GitHub Pages**.
