## 🎯 ما هو وقتك؟

**وقتك** هي أداة ويب متكاملة لمتابعة حسابات Bybit، تحسب تلقائياً:
- ⏱ **مرحلة 1:** موعد المراجعة الأولى (بعد 4 أيام من الإيداع)
- 🎯 **مرحلة 2:** موعد الجاهزية التامة (بعد 8 أيام من الإيداع)
- 📊 عرض العداد التنازلي الدقيق لكل مرحلة

كل البيانات محفوظة **محلياً** على جهازك — بدون سيرفر، بدون اشتراك، بدون إنترنت (بعد التحميل).

## ✨ المميزات الكاملة

### 🧮 الحاسبة السريعة
| الميزة | التفاصيل |
|--------|----------|
| UTC Format | يدعم تنسيق `2026-05-26 11:36:42 (UTC+0)` |
| Gmail Format | يدعم تنسيق `May 26, 2026, 2:36 PM` |
| Manual Input | إدخال يدوي بالتوقيت المصري |
| Progress Ring | حلقة تقدم بصرية مع النسبة المئوية |
| Live Countdown | عداد تنازلي يتحدث كل ثانية |
| Quick Examples | أمثلة سريعة لملء الحقول |
| Paste Button | زر لصق من الحافظة |

### 👥 مدير الحسابات
| الميزة | التفاصيل |
|--------|----------|
| إضافة حسابات | UID، البريد، IP، قيمة الإيداع، ملاحظات |
| تعديل الحسابات | تعديل أي حساب في أي وقت |
| حذف الحسابات | حذف فردي مع تأكيد |
| مرحلة 2 | زر لبدء المرحلة الثانية وتسجيل وقتها |
| بحث | بحث فوري بـ UID أو البريد الإلكتروني |
| فلترة | فلتر حسب المرحلة (1، 2، مكتمل) |
| ترتيب | ترتيب حسب الأحدث / الأقدم / الأقرب للانتهاء |

### 📊 لوحة الإحصاءات (Dashboard)
- إجمالي الحسابات
- الحسابات في المرحلة 1
- الحسابات في المرحلة 2
- الحسابات المكتملة
- إجمالي قيمة الإيداعات
- الحسابات المنتهية خلال 24 ساعة

### 📅 الجدول الزمني (Timeline)
- عرض بصري شريطي لجميع الحسابات
- تمييز مرحلة 1 ومرحلة 2 بألوان مختلفة
- مؤشر "اليوم" يظهر الموقع الحالي
- النقر على حساب يوصلك إلى بطاقته

### 🌐 IP Conflict Alert
- يكشف تلقائياً إذا كان نفس الـ IP مستخدماً في أكثر من حساب
- يعرض شارة تحذيرية `⚠️ IP مكرر` على البطاقة
- يُنبه عن خطر ربط الحسابات ببعضها

### 💾 نظام التخزين الثنائي
- **localStorage:** للحفظ الفوري والسريع
- **IndexedDB:** للتخزين المستمر وعدم الفقدان بعد الإغلاق
- **مزامنة تلقائية** بين المصدرين عند كل تغيير
- استعادة تلقائية من IDB إذا كانت localStorage فارغة

### 🔍 فحص سلامة البيانات
- مقارنة بين localStorage و IndexedDB
- عرض تقرير مفصل بالاختلافات
- زر مزامنة: نقل البيانات من localStorage إلى IDB
- زر استعادة: نقل البيانات من IDB إلى localStorage
- **إصلاح تلقائي** عند اكتشاف اختلاف

### 📤 تصدير / استيراد
- **تصدير JSON** — نسخة احتياطية كاملة
- **استيراد JSON** — استعادة البيانات من ملف
- **تصدير CSV** — جداول بيانات Excel-compatible

### ☑️ التحديد المتعدد (Bulk Mode)
- تحديد حسابات متعددة
- حذف جماعي مع تأكيد
- تصدير المحدد (JSON أو CSV)
- تحديد/إلغاء الكل

### 🔔 إشعارات المتصفح
- إشعار عند انتهاء المرحلة 1
- إشعار عند إكمال المرحلة 2
- يتطلب موافقة المستخدم

### 🌙 الوضع الداكن / الفاتح
- تبديل بين الوضعين بزر واحد
- يُحفظ التفضيل للزيارات القادمة
- اختصار `Ctrl+B`

### 🌍 ثنائي اللغة (عربي / إنجليزي)
- واجهة كاملة بالعربية (RTL) أو الإنجليزية (LTR)
- جميع الرسائل والتسميات مترجمة
- يُحفظ اختيار اللغة

### ⌨️ اختصارات لوحة المفاتيح
| الاختصار | الوظيفة |
|----------|---------|
| `Ctrl + Enter` | احسب / احفظ الحساب |
| `Ctrl + N` | حساب جديد (في مدير الحسابات) |
| `Ctrl + E` | تصدير البيانات |
| `Ctrl + B` | تبديل الوضع الداكن/الفاتح |
| `Ctrl + K` | فتح نافذة الاختصارات |
| `Ctrl + Shift + F` | بحث سريع |
| `Escape` | إغلاق النموذج |
| `?` | فتح نافذة الاختصارات |

### 📱 PWA (تطبيق قابل للتثبيت)
- يمكن تثبيته على الموبايل أو الكمبيوتر
- يعمل بدون إنترنت بعد أول تحميل
- Service Worker يخزن الأصول محلياً

## 🗂️ بنية البيانات (Account Object)

```json
{
  "id": 1716720002000,
  "uid": "12345678",
  "email": "example@gmail.com",
  "ip": "192.168.1.1",
  "amount": "100",
  "notes": "ملاحظات اختيارية",
  "depositTime": "2026-05-26T08:36:42.000Z",
  "stage": 1,
  "stage2StartTime": null,
  "auditLog": [
    { "action": "created", "time": "2026-05-26T11:00:00.000Z" }
  ]

```

### حقل `stage`
| القيمة | المعنى |
|--------|--------|
| `1` | في المرحلة الأولى (الانتظار 4 أيام) |
| `2` | في المرحلة الثانية (بعد الضغط على زر إتمام المهمة) |

## 🚀 التشغيل المحلي

لا يحتاج التطبيق أي build أو npm. فقط افتح `index.html` في المتصفح:

```bash
# عبر أي HTTP server مثل:
npx serve .
# أو
python -m http.server 3000
```

> ⚠️ **مهم:** لا تفتح `index.html` مباشرة كملف (`file://`) لأن IndexedDB وService Worker لا يعملان بدون HTTP server.

## 📌 ملاحظات تقنية

- **التوقيت:** كل الأوقات تُخزن بتوقيت UTC، وتُعرض بالتوقيت المصري (Africa/Cairo)
- **Gmail Format:** يُعامل كتوقيت مصري صيفي (UTC+3) ويُحول تلقائياً
- **الأمان:** لا يوجد أي إرسال للبيانات — كل شيء محلي 100%
- **IP Conflict:** يكشف تكرار الـ IP فقط، لا يمنع الحفظ
